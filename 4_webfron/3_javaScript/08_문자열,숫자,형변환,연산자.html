문자열(String 내장 객체)

        자바스크립트에는 문자열을 나타내는 객체 string이 존재하며
        문자열과 관련된 기본적인 함수를 제공

숫자(math 내장 객체)

        숫자(number) 타입 리터럴 표기법:
        12(정수), 3.14(실수), infinity, NaN (Not a Number)
        
        Math : 숫자 관련 함수 제공 js객체

문자열 -> 숫자 형변환

        parseInt() : 정수로만 변환
        parseFloat() : "정수" -> 정수 , "실수" -> 실수
        Number() :  ,,

동등|동일 비교 연산자

        동등 비교 연산자 ( == , !=)
        -> 값이 같으면 true(자료형 무시)
        ex)"1" == 1 == true
           "0" == 0 == false
        
        동일 비교 연산자( === , !== )
        -> 값과 자료형이 모두 같아야 true
