정규 표현식 객체 생성 및 확인

        1. const regExp = new regExp("정규표현식");
        2. const regExp = /정규표현식/;
    
       1. regExp.test(문자열) 
       - 문자열에 정규표현식과 일치하는 패턴이 있는 경우 true 
       - 없으면 false
       2. reqExp.exec(문자열) 
       - 문자열에 정규표현식과 일치하는 패턴이 있을경우 처음 매칭되는 문자열을 반환
       - 없으면 null

정규 표현식의 메타 문자

       문자열의 패턴을 나타내는 문자.
       문자마다 지정된 특별한 뜻이 담겨있다

       a(일반 문자열 ) : 문자열 내에 a라는 문자열이 존재하는지 검색
       [abcd] : abcd하나라도 일치하는 문자가 있는지 검색
       ^(캐럿) : 문자열의 시작을 의미
       $ : 문자열의 끝을 의미
       \w : 아무 글자 (공백,특문,한글X)
       \d (digit,숫자) : 숫자 10진법
       \s 공간
       [ㄱ-힣] : 모든 한글
       \ : 이스케이프문자


       * 수량 관련 메타 문자
       a{5} : a문자가 5개
       a{2,5} : 2개 이상 5개 이하
       a{n,} : n개 이상
       * : 0개 이상
       + : 1개 이상
       ? : 0 또는 1
       . : 칸
