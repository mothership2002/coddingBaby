변수 선언 방식

        var     :   변수 , 변수명 중복 가능 , 함수 레벨 scope
        let     :   변수 , 변수명 중복 불가 , 블럭 레벨 scope 
        const   :   상수 , 변수명 중복 불가 , 블록 레벨 scope
        
        1순위   :   const
        2순위   :   let
        3순위   :   var

자바스크립트의 자료형

        자바스크립트는 변수 선언 시 별도의 자료형을 지정하지 않는다.
        변수에 대입되는 값에 의해 자료형이 결정된다.
        - String
        - number
        - boolean
        - object -> array
        - function
        - undefined
        - null  //리터럴의 개념
