배열

        배열 : 변수 묶음

배열 관련 함수

        (Stack 구조 관련 함수)
        push(): 마지막 요소 추가
        pop(): 배열 마지막 요소 추출

        배열.indexOf("값") : 일치하는 값을 가진 요소의 index 반환
                            없으면 -1 반환

        배열.sort([정렬기준 함수]) : 배열 내 요소를 오름차순 정렬
                                    - 문자로 취급하기 때문
        배열.toString : 배열 요소 하나의 문자열로 출력
                        -,추가
        배열.join(구분자) : 배열 요소 하나의 문자열로 출력
                            -구분자 추가
