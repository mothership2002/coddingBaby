DOM(document object model)

        웹 문서의 모든 요소를 객체 형식으로 표현한 것
        - 문서 내 특정 요소에 접근방법이 용이

dom 사용 접근

        1.id로 접근하기 : document.getElementById("아이디");
        2.class로 접근 : document.getElementsClassName("클래스값");
        3.name으로 접근 : document.getElementsName("내임값");
        4.tag로 접근 : document.getElementsTagName("태그명");
        5.css선택자로 접근
            1) document.querySelector("css 선택자");
                -  선택된 요소가 여러 개 일 경우 , 첫번재 요소만 접근
            2) document.querySelectorAll("css 선택자");
                -  선택된 요소 모두 접근
