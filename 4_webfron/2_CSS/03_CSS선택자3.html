문자열 속성 선택자

        속성 값의 문자열을 확인하여 스타일 적용
        1) 선택자[속성명~="특정값"]{css}
        띄어쓰기 구분
        /<.div class="test-1 div-test"> 

        2) 선택자[속성명|="특정값"]{css}
        -기호로 구분

        3) 선택자[속성명^="값"]{}
        값으로 시작하는 태그

        4) selector[property $= "value"]{}
        값으로 끝나는 태그

        5) selecto[ ~ *= "~"]{}
