화면 배치 방법(형식) : display 속성

        요소가 화면이 어떻게 보여질지 형식을 지정하는 속성
        - block : 화면을 수직 분할
                  + width / height 속성 O
        - inline : 화면을 수평 분할 
                  + width / hight 속성 사용 X

        - inline-block : inline의 수평 분할 + block의 크기 조정
        
        - none : 화면에 요소가 보이지 않을 뿐 존재 중

        - flex : 요소의 정렬되는 방향, 요소 간의 간격을 유연하게 처리하는 형식
