영역 분할 형식

        
block 형식

            수직화면 분할
            width height 사용 가능

            ex) p/pre, h1~h6, hr, div  
    
        
inline 형식

            수평화면 분할
            width height 사용 불가능

            ex) strong/b (글자 굵게) ,em/i (기울기), mark,span(줄 분할)

  • inline : 내부에 컨텐츠 단위로 영역 지정
