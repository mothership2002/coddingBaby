04_표 관련 태그

        
table 태그

        웹 문서에서 자료 정리를 위해 사용
        행과 열로 이뤄짐 (cell)
    
        
tr 태그

        Table Row 
        한 행을 나타내는 태그
    
        
td 태그

        Table Date 
        한 셀을 나타내는 태그
    
        
th 태그

        Table Header 
        추가적으로 굵은 글씨, 가운데 정렬
        기능은 td태그와 같음
    
        
caption 태그

        Table caption for title or explanation

Edge MS https://www.microsoft.com
FireFox Mozila https://www.Mozila.org
